# drc

A Rust libary to introduce the `Drc` smart pointer along with its `Weak` counterpart.

Documentation (needs link)

## Usage

Add the following to your `Cargo.toml`:

```toml
[dependencies]
drc = "0.1"
```

and then add this to your crate root:

```rust
extern crate drc;
```
